## 参数说明

### VerticalTab 参数

| 参数    | 说明               | 类型    | 可选值               | 默认值    |
| ------- | ------------------ | ------- | -------------------- | --------- |
| height  | 高度               | number  | 'full'(代表撑满屏幕) | -         | - |
| tabs    | 标签页数组         | tabs[]  | -                    | []        |
| current | 默认激活的标签页   | string  | -                    | -         |
| backTop | 点击标题回到最上方 | boolean | -                    | _`false`_ |

### tabs

| 参数 | 说明       | 类型   | 可选值 | 默认值 |
| ---- | ---------- | ------ | ------ | ------ |
| id   | _Tab id_   | string | -      | -      |
| name | _Tab 名称_ | string | -      | -      |
