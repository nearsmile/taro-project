## 使用说明

```jsx
// 按需引入需要在 app.scss 中引入对应样式 screenDrawer.scss
import { ClScreenDrawer } from "mp-colorui";
```

## 参数说明

### ScreenDrawer 参数

| 参数         | 说明                | 类型    | 可选值             | 默认值    |
| ------------ | ------------------- | ------- | ------------------ | --------- |
| show         | _显示 screenDrawer_ | boolean | _`true`_,_`false`_ | _`false`_ |
| renderPage   | 页面显示内容        | Element | -                  | -         |
| renderDrawer | 抽屉显示的内容      | Element | -                  | -         |

### ScreenDrawer 事件

| 事件名称 | 说明                 | 参数返回 |
| -------- | -------------------- | -------- |
| onHide   | _点击返回小箭头触发_ | -        |
